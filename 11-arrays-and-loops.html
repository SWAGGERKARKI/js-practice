Arrays and loops

Array:

  • Another type of value in JavaScript.
  • It represents list of values.

For Example:

  • [10, 20, 30]
  • [10, "Sagar", true, { name: "Firos" }, [1, 2, 3]]

If we check typeof of array then it gives 'object'. Though it is object, it has its properties and method. To check an array we can use method, Array.isArray()

Access array values

  • By using bracket notation: array[index]

Properties and Methods:

  • length property: array.length
  • push method: array.push(value) - adds value to the last of array
  • splash method: array.splice(start-index, quantity) - removes specific values from array

Loops

  • While loop: prefer for non-standard loop (where all parts of the loop should not be defined)
  • For loop: prefer for standard loop (where all parts of the loop should be defined)

Parts in loop statement(control statement)

  • loop variable
  • loop condition
  • increment step
